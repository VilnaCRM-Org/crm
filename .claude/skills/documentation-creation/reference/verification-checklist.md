# Documentation Verification Checklist

- [ ] Referenced files exist or are clearly proposed new files.
- [ ] Make targets exist in `Makefile`.
- [ ] Code fences include languages.
- [ ] No backend-only commands are used for frontend docs.
- [ ] Markdown passes `make lint-md`.
- [ ] Workflow docs mention `make format` before `make lint` for code changes.
- [ ] Agent skill docs keep BMAD references under `.agents/skills`.
